# poLite Compiler 🎩☕

Welcome to the **poLite** compiler repository! 

This project implements a custom compiler for **poLite**, a courteous and well-mannered variation of a C language subset. In poLite, good manners matter: instead of harsh keywords, you'll find gentle syntax (like using `do this ... pls` instead of standard `if` conditions). 

This project was developed for the *Formal Languages and Compilers* course and features:
- **Lexical Analysis:** Token recognition using FLEX.
- **Syntax Analysis:** Grammar parsing implemented with YACC.
- **Semantic Analysis:** A dynamic Symbol Table and a custom Type Checker that gracefully informs you of any type mismatches.
